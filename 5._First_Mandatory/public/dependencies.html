Dependencies

Dependencies bliver skrevet i package.json filen, som indeholder en samling af et projekts dependencies. Disse dependencies er de moduler, som projektet er afhængig af for at fungere korrekt.

package.json filen

  1. Ethvert projekt, som bruger NodeJs, vil få brug for en package.json fil, hvori der bl.a. står de dependencies og scripts, som projektet har brug for, for at kunne køre.
  2. Dependencies som vi har lært at bruge:
  3. Express
  4. DevDependencies


Hvordan man installerer Express

  1. Gå ind i din foretrukne terminal og skriv flg: node -v
  2. Hvorefter den sender NodeJS versionen tilbage, hvis den er installeret
Hvad er npm?

  1. Det er et online repository til at publicering af open-source Node
  2. Det er et kommandolinjeværktøj til interaktion med dette repository, der hjælper med pakkeinstallation, versionstyring og afhængighedsstyring
  3. Det bliver installeret, når man installerer NodeJS
  4. Man kan teste, om den er der ved at skrive flg. i terminalen: npm -v
  5. Hvorefter den sender npm versionen tilbage
